 PDF 도구 상자 — 사용법
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

■ 실행
  바탕화면의 "PDF 도구 상자" 바로가기를 더블클릭하거나
  이 폴더의 run.bat 을 실행하세요.

■ 기능
  1. PNG로 변환      : PDF 페이지들을 PNG 이미지로 저장
  2. PDF로 추출      : 원하는 페이지만 골라 새 PDF 생성
  3. 텍스트 / OCR    : 텍스트 추출. 스캔본 페이지는 자동 OCR.
                       PNG/JPG 이미지를 선택해도 바로 OCR 됩니다
  4. 이미지 → PDF    : PNG/JPG 여러 장을 PDF 한 개로 합치기

■ 페이지 선택
  "페이지" 입력칸에 아래처럼 입력:
    전체          → 모든 페이지
    1-3           → 1,2,3 페이지
    1-3,5,8       → 1,2,3,5,8 페이지

■ 결과 저장 위치
  원본 파일과 같은 폴더에 "파일명_output" 폴더가 생기고
  그 안에 결과물이 저장됩니다. 완료되면 폴더가 자동으로 열립니다.

■ OCR (스캔본 PDF에서 글자 읽기) — 선택 설치
  스캔본 PDF의 텍스트를 읽으려면 Tesseract 설치가 필요합니다.
  (텍스트가 이미 있는 PDF는 설치 없이 바로 추출됩니다)

  1) 아래 주소에서 설치 파일 다운로드:
     https://github.com/UB-Mannheim/tesseract/wiki
     (tesseract-ocr-w64-setup-*.exe)
  2) 설치 중 "Additional language data" 에서
     ★ Korean 을 반드시 체크! (한글 인식용)
  3) 설치 후 앱을 다시 실행하면 자동으로 인식됩니다.

■ 문제가 생기면
  - 라이브러리 오류: setup.bat 을 한 번 실행
  - OCR 결과가 [OCR 불가]로 나옴: 위의 Tesseract 설치 필요
